HTML

HTML (HyperText Markup Language) is the foundational language used to create web pages and web applications. It describes the structure of a web page and its content, which can include text, images, links, forms, and other multimedia elements. HTML elements are the building blocks of HTML pages, and they are represented by tags.

Heading

The heading tags in HTML are used to define headings for web pages. There are six levels of headings, from h1 to h6, with h1 being the highest level (or most important) and h6 being the lowest level (or least important)

Paragraph

In HTML, the p tag is used to define a paragraph. Paragraphs are blocks of text that are separated from other blocks by some space, providing structure and readability to the content. The p tag automatically adds some margin before and after the paragraph to create separation from other elements.
  • The "p" tag is used to define paragraph

Formating tags

Formatting tags in HTML are used to style and format text, making it more readable and visually appealing. These tags allow you to apply specific styles such as bold, italic, underline, and more to parts of your text content. Here are some common formatting tags.
  1. Bold:It makes the text bold
  2. Italic:Makes the text italic
  3. Strong: Which tells the browser that text is important
  4. Underline:It is used to underline the text
  5. Mark: This tag is used to highlights the text
  6. del:This tag is used to display the deleted content
  7. Superscript: Rises the text above the normal line
  8. Subscript:Lowers the text slightly below the line

Image-tag

  • The tag in HTML is used to embed images in a webpage. It is an inline element that does not have a closing tag.
  • The tag requires specific attributes to properly display the image and provide essential information for accessibility and SEO
  • Images are not technically inserted into a web page
  • src - Specifies the path to the image
  • alt - Specifies an alternate text for the image, if the image for some reason cannot be displayed

Anchor tag

  1. Link Name
  2. The HTML element (or anchor element), with its href attribute
  3. creates a hyperlink to web pages, files, email addresses, locations in the same page, or anything else a URL can address.
  4. This attribute determines where the user is directed upon clicking the link.

Structure of URl

  • A URL consists of ten parts: the scheme, subdomain, top-level domain, second-level domain, subdirectory, parameter, port, path, query, and fragment
  • https://www.example.com/page.html
  • The four main components of URLs are the protocol, domain, path, and query.
  • There are two main types of URLs: absolute and relative.
